# Git Teaching Examples

ชุดนี้จัดตามลำดับสไลด์ในโฟลเดอร์ `git/` เพื่อใช้สอน Git แบบเป็นเรื่อง ๆ

- `index.html` คือหน้าเริ่มต้น
- `topics/` คือ lab แยกตามหัวข้อในสไลด์
- `LABS.md` คือโจทย์ฝึก Easy/Medium/Hard ตามลำดับบทเรียน
- `LAB_SOLUTIONS.md` คือเฉลยแนวทางสำหรับโจทย์ฝึก
- `SLIDE_COVERAGE.md` คือแผนที่ว่า slide topic ไหนอยู่ใน lab ไฟล์ใด
- `notes/` คือบันทึกประกอบการสอนรายบท

## วิธีใช้สอน

1. เปิด `index.html`
2. เข้า `topics/index.html`
3. เลือก lab ตามหัวข้อที่กำลังสอน
4. เปิด `LABS.md` เพื่อให้ผู้เรียนทำโจทย์ฝึก
5. ให้ผู้เรียนพิมพ์ command ตาม lab ใน repo ทดลองของตัวเอง
6. ใช้ `LAB_SOLUTIONS.md` ตรวจงานหรือให้ hint
7. ใช้ `SLIDE_COVERAGE.md` ตรวจว่าหัวข้อใน slide มีตัวอย่างรองรับครบ

## หมายเหตุ

Lab เป็นตัวอย่างคำสั่ง Git สำหรับสอน ไม่ควรรันใน repository งานจริง ให้สร้างโฟลเดอร์ทดลองแยกก่อนเสมอ
