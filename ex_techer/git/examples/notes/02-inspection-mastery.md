# 02 - Git Inspection Mastery

## เป้าหมาย

ให้นักเรียนตรวจสอบประวัติและความเปลี่ยนแปลงได้ก่อน commit, หลัง commit และระหว่าง branch

## Flow การสอน

1. ใช้ `git log` เพื่อดู timeline
2. ใช้ filter เช่น `--since`, `--author`, `--grep`
3. อ่าน commit ด้วย `git show`
4. เปรียบเทียบพื้นที่ด้วย `git diff`, `git diff --staged`, `git diff HEAD`
5. เปรียบเทียบ commit/branch
6. สรุป command matrix ว่าสถานการณ์ไหนควรใช้คำสั่งไหน
