# 04 - Git Remotes

## เป้าหมาย

ให้นักเรียนเข้าใจการทำงานร่วมกันผ่าน remote repository และใช้ clone, fetch, pull, push ได้ถูกสถานการณ์

## Flow การสอน

1. อธิบาย local vs remote repository
2. Clone repo และตรวจ remote ด้วย `git remote -v`
3. เปรียบเทียบ `git fetch` กับ `git pull`
4. ส่งงานด้วย `git push origin main`
5. ใช้ feature branch flow สำหรับทีม
6. สรุป daily collaboration loop
