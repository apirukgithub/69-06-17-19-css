# 03 - Git Branching and Merging

## เป้าหมาย

ให้นักเรียนเข้าใจ branch เป็น pointer, ทำ feature branch ได้, merge ได้ และแก้ conflict ได้

## Flow การสอน

1. อธิบาย branch pointer และ HEAD
2. ใช้ `git branch` เพื่อ list/create/delete
3. ใช้ `git switch` สำหรับสลับ branch
4. ทำ feature lifecycle: create -> work -> commit -> return -> merge
5. เปรียบเทียบ fast-forward กับ three-way merge
6. สร้าง conflict จำลองและแก้ด้วย conflict markers
7. ใช้ `git stash` เมื่อมีงานค้าง
