02.4 Comparing Commits and Branches

Compare patterns

$ git diff HEAD~1 HEAD
$ git diff main feature
$ git diff main..feature
$ git log main..feature --oneline
$ git log --left-right --graph main...feature

ใช้สอนความต่างของ two-dot และ three-dot ในการดู history/branch comparison
