01.3 Add, Commit and File States

ฝึกอ่าน state ของไฟล์ระหว่าง untracked, modified, staged และ committed

BackPreviousNext

State Matrix

StateTracked?LocationNext action
UntrackedNoWorking Directory`git add`
ModifiedYesWorking Directory`git add`
StagedYesStaging Area`git commit`
CommittedYesRepositoryแก้ต่อหรือดู history

Commands

$ echo "first" > story.txt
$ git status --short
$ git add story.txt
$ git status --short
$ git commit -m "Add story"
$ echo "second" >> story.txt
$ git status --short
