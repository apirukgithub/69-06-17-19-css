02.3 Diff: Working Tree, Index and HEAD

เลือกคำสั่ง diff ให้ตรงกับพื้นที่ที่อยากเปรียบเทียบ

BackPreviousNext

What should I type?

อยากดูอะไรCommandเปรียบเทียบ
ไฟล์ที่แก้แต่ยังไม่ add`git diff`Working Tree vs Index
สิ่งที่ add แล้วรอ commit`git diff --staged`Index vs HEAD
ทั้งหมดตั้งแต่ commit ล่าสุด`git diff HEAD`Working Tree vs HEAD

Lab

$ echo "line 1" > app.txt
$ git add app.txt
$ git commit -m "Add app"
$ echo "line 2" >> app.txt
$ git diff
$ git add app.txt
$ git diff --staged
$ echo "line 3" >> app.txt
$ git diff HEAD
