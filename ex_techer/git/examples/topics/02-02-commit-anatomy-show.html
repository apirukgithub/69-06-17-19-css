02.2 Commit Anatomy and git show

อ่าน commit hash, author, date, message และ patch

BackPreviousNext

Commit anatomy

  • SHA-1 hash: ตัวระบุ commit
  • Author: คนสร้าง commit
  • Date: เวลา
  • Message: คำอธิบาย
  • Patch: สิ่งที่เปลี่ยน

Commands

$ git log --oneline
$ git show 
$ git show --stat 
$ git show --name-only
