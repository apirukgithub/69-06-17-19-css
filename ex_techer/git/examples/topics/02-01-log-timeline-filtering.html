Useful log views

$ git log
$ git log --oneline
$ git log --oneline --graph --decorate
$ git log --since="2 weeks ago"
$ git log --author="Alice"
$ git log --grep="login"
สอนให้เริ่มจาก `--oneline --graph --decorate` เพราะอ่าน branch/history ได้ง่ายกว่าหน้า log เต็ม
