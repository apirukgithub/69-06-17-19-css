Branch commands

$ git branch
$ git branch feature/login
$ git switch feature/login
$ git branch -d feature/login
$ git branch -D feature/login
ใช้ `-d` เมื่อต้องการลบแบบปลอดภัย และ `-D` เมื่อต้อง force delete
