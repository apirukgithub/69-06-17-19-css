Repeatable workflow

  1. แก้ไฟล์ใน Working Directory
  2. ตรวจด้วย `git status`
  3. เลือกไฟล์ด้วย `git add`
  4. บันทึก snapshot ด้วย `git commit`
  5. ตรวจ repo clean ด้วย `git status`
# edit files
$ git status
$ git add .
$ git commit -m "Describe the change"
$ git status
