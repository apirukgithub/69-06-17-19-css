03.3 Merge: Fast-forward and Three-way

แยกสถานการณ์ merge ที่ Git เลื่อน pointer ได้กับต้องสร้าง merge commit

BackPreviousNext

Fast-forward

$ git switch main
$ git merge feature/simple

เกิดเมื่อ main ไม่มี commit ใหม่หลังแตก branch

Three-way merge

$ git switch main
$ git merge feature/complex

เกิดเมื่อทั้ง main และ feature มี commit ใหม่ ต้องสร้าง merge commit
