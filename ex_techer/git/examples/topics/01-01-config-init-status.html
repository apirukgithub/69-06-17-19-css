01.1 Config, Init and Status

ตั้งค่าตัวตน สร้าง repository และอ่านสถานะไฟล์

BackNext

Concept

ก่อนเริ่มใช้ Git ต้องมี identity และเข้าใจว่า repo เริ่มจากโฟลเดอร์ธรรมดาที่ถูก `git init` เปลี่ยนให้มี `.git`

  • Working Directory: ไฟล์ที่กำลังแก้
  • Staging Area: คิวสำหรับ commit
  • Repository: ประวัติ snapshot

Commands

$ git config --global user.name "Your Name"
$ git config --global user.email "you@example.com"
$ mkdir git-lab
$ cd git-lab
$ git init
$ git status

Exercise

สร้างไฟล์ `README.md` แล้วรัน `git status` เพื่อดูว่าไฟล์อยู่ในสถานะ untracked
