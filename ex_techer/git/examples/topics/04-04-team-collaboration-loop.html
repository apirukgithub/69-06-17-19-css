Daily loop

  1. รับงานใหม่: `git pull` หรือ `git fetch` แล้วตรวจ
  2. สร้าง branch: `git switch -c feature/name`
  3. commit งานเล็ก ๆ: `git add` + `git commit`
  4. อัปเดตก่อนส่ง: `git fetch`/`git pull`
  5. ส่งงาน: `git push -u origin feature/name`
$ git pull origin main
$ git switch -c feature/profile-card
# edit, add, commit
$ git push -u origin feature/profile-card
